# Air Hockey Game – React Native

## Description
A two-player Air Hockey game developed using React Native.
Supports multiple game modes including timed matches and first-to-10 gameplay.

## Features
- Two-player local gameplay
- Draggable paddles
- Animated puck physics
- Scoreboard and win detection
- Restart game option
- Clean UI with center markings

## Tech Stack
- React Native
- TypeScript
- Animated API
- PanResponder

## How to Run
npm install
npx react-native run-android

## Emulator
Pixel 5 – Android 13
